# C Programming Course Repository
This repository contains a step-by-step curriculum from Beginner to Advanced in the C Programming Language. 
Each folder contains the lesson explanation and code. 

## Curriculum Topics:
- 01_hello_world
- 02_data_types_and_variables
- 03_operators
- 04_control_flow
- 05_functions
- 06_arrays_and_strings
- 07_pointers_basics
- 08_dynamic_memory
- 09_structures_and_unions
- 10_advanced_pointers
- 11_file_handling
